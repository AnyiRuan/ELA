## 🔐 访问管理后台

### 1. 找到管理员入口

在网站首页底部，您会看到一个小的"管理员入口"链接。点击它即可进入管理后台。

### 2. 登录

- **默认密码**: `ela2024admin`
- **重要**: 首次部署后，请在 Vercel 环境变量中修改 `ADMIN_PASSWORD` 为更安全的密码

#### 如何修改密码：

1. 登录 [Vercel Dashboard](https://vercel.com)
2. 选择您的项目
3. 进入 **Settings** → **Environment Variables**
4. 找到 `ADMIN_PASSWORD` 并修改为新密码
5. 重新部署项目使更改生效

## 📝 网页内容管理

管理后台的第一个标签页是"网页内容管理"，这里可以修改网站首页的主要内容。

### 可以管理的内容：

#### 中文内容
1. **首页标题** - 网站顶部的主标题
2. **首页描述** - 主标题下方的描述文字
3. **差异化优势标题** - "Why ELA" 部分的标题
4. **差异化优势副标题** - 优势说明文字
5. **服务描述** - 三个主要服务的详细描述
   - 商业领袖导师团队
   - 真实商业沉浸式体验
   - 学术规划与申请

#### English Content
所有对应的英文内容也可以在这里管理。

### 使用方法：

1. 点击顶部的语言切换按钮（中文/English）
2. 在表单中输入或修改内容
3. 点击底部的"保存内容更改"按钮
4. 等待"✓ 内容已成功保存！"的提示

### 💡 编辑建议：

- **标题**：简洁有力，控制在 30 字以内
- **描述**：清晰明了，建议 100 字左右
- **服务描述**：突出特色和优势，200-300 字为佳

## 🤖 聊天机器人知识库管理

管理后台的第二个标签页是"聊天机器人知识库管理"，这里可以定制 AI 助手的回答内容。

### 知识库包含的部分：

#### 1. 聊天机器人身份介绍
定义机器人的角色和身份，例如：
```
你是ELA English的智能客服助手。ELA是一家专业的教育咨询机构...
```

#### 2. 核心服务介绍
列出所有服务项目，机器人会根据这些内容回答用户问题：
```
我们的服务包括：
- 商业领袖导师团队：...
- 真实商业沉浸式体验：...
- 学术规划与申请：...
```

#### 3. 团队特色
描述团队的独特之处：
```
我们的团队特色：核心团队由成功企业家组成...
```

#### 4. 回答风格指导
告诉机器人如何回答问题：
```
请用友好、专业的语气回答用户问题。保持简洁明了...
```

### 使用方法：

1. 点击语言切换按钮（中文知识库/English Knowledge Base）
2. 编辑相应的文本框
3. 点击"保存知识库更改"按钮
4. 等待保存成功提示

### 💡 编辑建议：

- **身份介绍**：清晰定义机器人的角色
- **服务介绍**：详细但不冗长，每项服务 50-100 字
- **团队特色**：突出差异化优势
- **回答风格**：明确语气（友好、专业、简洁等）

## ⚠️ 重要注意事项

### 数据存储说明

**当前版本使用临时存储**，这意味着：

- ✅ 您的更改会立即生效
- ⚠️ 在 Vercel 重新部署或冷启动后，更改可能会丢失
- 📝 建议保存一份内容备份到本地文档

### 未来升级建议

为了实现永久存储，建议配置以下服务之一：
- Vercel KV (键值存储)
- Vercel Postgres (数据库)
- 其他云数据库服务

## ❓ 常见问题

### Q: 我修改的内容什么时候生效？
**A:** 立即生效！保存后刷新网站页面即可看到更改。

### Q: 我可以恢复到之前的内容吗？
**A:** 建议在修改前复制原内容到记事本保存。当前版本不支持版本历史。

### Q: 聊天机器人的回答不准确怎么办？
**A:** 在"聊天机器人知识库"中完善服务介绍和回答风格指导，提供更详细的信息。

### Q: 我忘记了管理员密码怎么办？
**A:**
1. 登录 Vercel Dashboard
2. 进入项目设置
3. 查看或重置 `ADMIN_PASSWORD` 环境变量

### Q: 可以添加更多的可编辑字段吗？
**A:** 可以！如需添加更多字段，请联系技术支持或参考开发文档。

### Q: 内容保存后为什么看不到变化？
**A:** 请检查：
1. 是否点击了保存按钮
2. 是否看到成功提示
3. 是否刷新了网站页面
4. 对于知识库，需要重新打开聊天窗口

### Q: 我可以同时编辑中文和英文内容吗？
**A:** 可以！切换语言标签后分别编辑，最后一起保存即可。
